## install

Download latest (debs)[http://noone.org/conkeror-nightly-debs] and install via dpkg -i.


### custom
add web-jump ddg as alias of duckduckgo in the .conkerorrc

webjumps.g = webjumps.google;
webjumps.ddg = webjumps.duckduckgo;
